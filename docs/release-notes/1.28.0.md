## Istio Updated to Version 1.30.2

We've updated the Istio version to 1.30.2 ([#2124](https://github.com/kyma-project/istio/pull/2124)). See [Announcing Istio 1.30.2](https://istio.io/latest/news/releases/1.30.x/announcing-1.30.2/).

## New Experimental Features

- We've added support for Vertical Pod Autoscaler (VPA) for Istio control plane components (istiod, ingress gateway, egress gateway, CNI).
See [Alpha Istio Features](../user/00-45-istio-features-configmap.md). See [#2011](https://github.com/kyma-project/istio/issues/2011).

## Fixed Bugs

- We've fixed an issue where Gardener was not properly detected.
- We've improved reconciliation error handling by distinguishing between warning and error states. Warning-level errors now trigger graceful requeuing with a fixed interval, while error-level errors continue to use the default exponential backoff. See [#2112](https://github.com/kyma-project/istio/pull/2112).
